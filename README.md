# Based

- React
- Flask

# Dependency libraries

## Front-end

- antd(Ant Design of React)

## Back-end

- flask
- flask-cors
- pandas

# How to Start  

## Front-end command:
```
    cd Web
    yarn install
    yarn dev
```

then open the following url in browser: http://localhost:3000/

## Back-end command:
```
    cd API
    pip install -r req.txt
    python main.py
```
